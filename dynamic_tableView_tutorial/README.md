# 다이나믹 테이블 뷰 

## 공부이유 : 오토레이아웃과 테이블뷰를 컨트롤함에 있어 부족함을 느껴 공부를 다시해보기로 함
#### 배운점: 기존에는 스토리보드 하나에 뷰컨트롤러에 테이블뷰부터 셀까지 모두 만든 후 작업을 하였다. <br>이번에는 nib을 이용하여 셀에 대한 디자인을 해보았고 스택뷰의 아이템을 일정하게 맞추는법을 추가로 알게되었다
